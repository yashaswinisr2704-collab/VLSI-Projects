# Verilog HDL Project

## Description
This repository contains a Verilog HDL implementation of a digital circuit along with its testbench and FPGA constraint file. The project was developed for learning and practicing RTL design concepts.

## Files
- `design.v` – Verilog HDL source code for the digital circuit.
- `testbench.v` – Testbench used to verify the functionality of the design through simulation.
- `constraints.xdc` – FPGA constraints file containing pin assignments and timing constraints (if applicable).

## Tools Used
- Verilog HDL
- EDA Playground
- Vivado (for synthesis and implementation, if used)

## Features
- RTL design implementation
- Functional verification using a testbench
- FPGA constraints included for hardware implementation

## How to Run
1. Open the project in EDA Playground or Vivado.
2. Add `design.v` and `testbench.v`.
3. Select a Verilog simulator (e.g., Icarus Verilog or XSIM).
4. Run the simulation and verify the output waveforms.
